<h2>📌 Overview</h2>

**NeuroVision AI** is an intelligent Chrome Extension designed to improve digital accessibility for neurodivergent users.

The extension uses AI to:

* simplify difficult content
* summarize long articles
* explain complex text
* reduce cognitive overload
* improve focus while browsing

NeuroVision AI transforms overwhelming web content into clear, easy-to-understand information in real time.

---

<h2>🚀 Features</h2>

## 🧠 AI Text Simplification

Converts complex paragraphs into simple, readable language.

---

## 📖 Smart Summarization

Generates short summaries for lengthy articles and webpages.

---

## 💡 Context-Aware Explanations

Explains selected text in a beginner-friendly format.

---

## 🎯 Focus Assistance

Helps users focus only on important content and reduces distractions.

---

## 🌐 Works Across Websites

Runs directly inside Google Chrome and supports most websites.

---

## ⚡ Real-Time AI Responses

Uses Gemini API for fast and intelligent responses.

<h2>⚙️ Installation Guide</h2>

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/NeuroVision-AI.git
```

---

## 2️⃣ Open Chrome Extensions

Go to:

```bash
chrome://extensions/
```

---

## 3️⃣ Enable Developer Mode

Enable the **Developer Mode** toggle at the top-right corner.

---

## 4️⃣ Load Extension

Click:

```bash
Load Unpacked
```

Then select the project folder.

---

<h2>🔑 Gemini API Setup</h2>

## Step 1

Visit:

[https://aistudio.google.com/](https://aistudio.google.com/)

---

## Step 2

Generate your Gemini API Key.

---

## Step 3

Open:

```javascript
background.js
```

Replace:

```javascript
const GEMINI_API_KEY = "YOUR_API_KEY";
```

with your actual API key.
